# Credit Scoring Service (ML & API)

Интерактивный микросервис для оценки кредитоспособности заемщиков. Проект охватывает полный цикл: от предобработки данных и обучения модели XGBoost до развертывания REST API.

## Основные возможности
- **Real-time Inference:** Мгновенное предсказание вероятности дефолта через API.
- **Robust Validation:** Строгая валидация входящих данных с помощью Pydantic.
- **Model Interpretation:** Анализ ключевых факторов, влияющих на решение банка.

## Технологический стек
- **Backend:** FastAPI, Uvicorn.
- **Machine Learning:** XGBoost, Scikit-learn.
- **Data Engineering:** Pandas, Joblib.

## Результаты и метрики
В качестве основной модели использован **XGBoost**, показавший отличные результаты на тестовой выборке:
- **ROC-AUC:** 0.94
- Модель сохранена в файл `best_credit_model.pkl` для использования в продакшене.

## Установка и запуск
1. Установите зависимости: `pip install -r requirements.txt`
2. Запустите сервер: `python -m uvicorn app:app --reload`
3. Перейдите на `http://127.0.0.1:8000/docs` для тестирования.
